# L3-M4-S04 架构决策表达模板

## 一句话结论

- 架构决策表达模板 是 L3 阶段的关键能力点，面试回答建议覆盖“定义、原理、场景、边界”。

## 结构图

```mermaid
flowchart LR
    A[结论先行] --> B[原理展开]
    B --> C[项目场景]
    C --> D[边界追问]
```

## 核心知识点

1. 先给一句话结论，再展开关键点。
2. 回答要带项目场景和数据，不只背定义。
3. 提前准备追问分支，避免被问深后失焦。

## 高频面试题

### Q1：你如何在项目中落地“架构决策表达模板”？

答题骨架：
1. 先说明业务目标和约束。
2. 再给可执行方案和关键指标。
3. 最后补充风险、边界与回退策略。

### Q2：架构决策表达模板 的常见误区是什么？

答题骨架：
1. 说明常见错误做法。
2. 给出正确实践和适用条件。
3. 用一个真实场景收尾。

- 示例代码：本节以口述与流程训练为主，可结合已有示例复习。
