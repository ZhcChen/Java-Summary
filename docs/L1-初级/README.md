# L1 初级：Java 基础能力（书籍版目录）

> 这一层不追求“背答案”，目标是建立稳定的基础认知和可复用的编码习惯。

## 你将获得什么

完成 L1 后，应达到以下能力：

1. 能独立完成基础 CRUD 接口。
2. 能解释常见 Java 基础概念，不只会背术语。
3. 能写出带边界处理的基础代码（判空、异常、并发安全意识）。
4. 能对基础问题给出结构化表达（结论 -> 原理 -> 场景 -> 风险）。

## 阅读方式（建议）

- 第一次学习：按“模块顺序”从 01 到 04。
- 第二次复习：按“子章节索引”查漏补缺。
- 临近面试：优先看 04 模块，再回看 01~03 的薄弱点。

## 学完 L1 的验收标准

- 能在 10 分钟内讲清 `HashMap` 与 `ConcurrentHashMap` 的核心差异。
- 能说清 `synchronized` 与 `volatile` 各自解决的问题。
- 能画出一个最小后端请求链路（Controller -> Service -> DB/Cache）。
- 能完成至少 3 个可运行 Java 示例并解释关键注释。
