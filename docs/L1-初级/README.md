# L1 初级索引：基础夯实

## 阶段目标

- 能独立完成常见后端接口开发。
- 能清楚回答基础面试题（概念 + 原理 + 场景）。

## 阅读顺序

```mermaid
flowchart LR
    A[Java 语法与集合] --> B[并发基础]
    B --> C[JVM 入门]
    C --> D[Spring Boot 基础]
    D --> E[MySQL 基础]
    E --> F[Redis 基础]
```

- 每个模块至少 1 张图 + 3 道问答 + 1 个示例。
