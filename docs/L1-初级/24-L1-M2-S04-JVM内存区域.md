## 前置知识

- 知道对象和方法调用。

## 术语解释（零基础友好）

- **堆**：存放对象实例的主要区域。
- **栈**：存放方法栈帧和局部变量。
- **元空间**：JDK8+ 中存放类元数据的区域。

## 详细学习步骤（从不会到会）

1. 先通过代码区分栈变量和堆对象。
2. 再理解类信息加载后进入元空间。
3. 结合常见异常（堆 OOM/栈溢出）建立故障映射。
4. 最后画出 JVM 内存结构图并口述。

## 常见错误与纠偏

- 把所有数据都认为在堆里，导致排障方向错误。
- 只背概念，不会结合异常现象分析。

## Java 示例代码（含注释，可直接运行）


**建议文件名：** `Main.java`  
**运行命令：** `javac Main.java && java Main`

**预期输出（示例）：**
```text

```

```java
public class Main {
    public static void main(String[] args) {
        int local = 1;               // 栈上局部变量
        byte[] bytes = new byte[256]; // 堆上对象
        // 类元数据位于方法区（JDK8+ 为元空间）
        System.out.println("local=" + local);
        System.out.println("heapSize=" + bytes.length);
    }
}
```
