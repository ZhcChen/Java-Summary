## Java 示例代码（含注释，可直接运行）

**建议文件名：** `Main.java`  
**运行命令：** `javac Main.java && java Main`

**预期输出（示例）：**
```text
local=1
heapSize=256
```

```java
public class Main {
    public static void main(String[] args) {
        int local = 1;               // 栈上局部变量
        byte[] bytes = new byte[256]; // 堆上对象
        // 类元数据位于方法区（JDK8+ 为元空间）
        System.out.println("local=" + local);
        System.out.println("heapSize=" + bytes.length);
    }
}
```
