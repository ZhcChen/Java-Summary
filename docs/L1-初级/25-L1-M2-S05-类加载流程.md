## 前置知识

- 知道类和对象关系。
- 会写 `static` 代码块。
- 了解继承基础语法。

## 术语解释（零基础友好）

- **类加载**：把类字节码读入 JVM 并转为运行时结构。
- **初始化**：执行静态变量赋值与静态代码块。
- **双亲委派**：类加载器优先委托父加载器尝试加载。

## 详细学习步骤（从不会到会）

1. 写父子类静态块和实例块，观察实际输出顺序。
2. 区分“加载/链接/初始化”三个阶段职责。
3. 理解双亲委派解决的类冲突与安全问题。
4. 把流程转化为可口述的四步答题模板。

## 常见错误与纠偏

- 误以为 new 对象才会触发所有类加载。
- 混淆静态初始化与实例初始化时机。

- 父类静态通常先于子类静态执行。
- 主动使用类（new/访问静态成员等）会触发初始化。

## Java 示例代码（含注释，可直接运行）


**建议文件名：** `Main.java`  
**运行命令：** `javac Main.java && java Main`

**预期输出（示例）：**
```text
A static
B static
A instance
B instance
```

```java
public class Main {
    static class A {
        static { System.out.println("A static"); }
        { System.out.println("A instance"); }
    }
    static class B extends A {
        static { System.out.println("B static"); }
        { System.out.println("B instance"); }
    }

    public static void main(String[] args) {
        // 首次主动使用时触发类加载与初始化
        new B();
    }
}
```
