# L1-M4-S01 基础题答题模板训练

## 一句话结论

- 基础题答题模板训练 是 L1 阶段的关键能力点，面试回答建议覆盖“定义、原理、场景、边界”。

## 结构图

```mermaid
flowchart LR
    A[结论先行] --> B[原理展开]
    B --> C[项目场景]
    C --> D[边界追问]
```

## 核心知识点

1. 先给一句话结论，再展开关键点。
2. 回答要带项目场景和数据，不只背定义。
3. 提前准备追问分支，避免被问深后失焦。

## 高频面试题

### Q1：你如何在项目中落地“基础题答题模板训练”？

答题骨架：
1. 先说明业务目标和约束。
2. 再给可执行方案和关键指标。
3. 最后补充风险、边界与回退策略。

### Q2：基础题答题模板训练 的常见误区是什么？

答题骨架：
1. 说明常见错误做法。
2. 给出正确实践和适用条件。
3. 用一个真实场景收尾。

## 学习动作

- 示例代码：本节以口述与流程训练为主，可结合已有示例复习。
- 复习时至少完成 3 次 60~90 秒口述训练。
- 对照 [`../13-面试题库编号与复习规则.md`](../13-面试题库编号与复习规则.md) 补齐表达。

## 复习检查

- [ ] 能在 90 秒内说明核心结论
- [ ] 能说明至少 1 个项目场景
- [ ] 能回答 1 个追问问题
