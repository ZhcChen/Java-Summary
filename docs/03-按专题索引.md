## 推荐用法

- **准备校招/初中级社招**：优先 Java + Spring + MySQL + Redis。
- **准备中高级社招**：重点 JVM + 并发 + MySQL 优化 + MQ + 分布式。
- **准备架构岗**：重点高可用、容量评估、一致性取舍与治理体系。
