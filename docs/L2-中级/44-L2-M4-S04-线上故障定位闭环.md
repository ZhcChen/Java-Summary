# L2-M4-S04 线上故障定位闭环

## 一句话结论

- 线上故障定位闭环 是 L2 阶段的关键能力点，面试回答建议覆盖“定义、原理、场景、边界”。

## 核心知识点

1. 可观测性目标是“快定位、快止血、快复盘”。
2. Logs/Metrics/Trace 要使用统一标识（traceId）串联。
3. 告警要防噪音，优先业务影响指标。

## 高频面试题

### Q1：你如何在项目中落地“线上故障定位闭环”？

答题骨架：
1. 先说明业务目标和约束。
2. 再给可执行方案和关键指标。
3. 最后补充风险、边界与回退策略。

### Q2：线上故障定位闭环 的常见误区是什么？

答题骨架：
1. 说明常见错误做法。
2. 给出正确实践和适用条件。
3. 用一个真实场景收尾。
