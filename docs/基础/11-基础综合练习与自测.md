# 11 基础综合练习与自测

## 本章目标

- 把前 10 章知识串成可执行的基础能力闭环。
- 通过综合题验证“理解 + 代码 + 表达”三项能力。
- 明确下一阶段（进阶）前需要补齐的短板。

## 前置知识

- 已完成基础层 01~10 章节学习。
- 能独立运行每章 Java 示例代码。

## 核心概念

1. 综合能力 = 概念正确 + 代码可运行 + 能解释取舍。
2. 自测不只看“能跑通”，还要看“是否可维护”。
3. 复盘要落到行动项，避免重复犯错。

## 原理展开

- 基础阶段最常见问题是“章节会了，但串不起来”。
- 综合题通过统一场景（如用户查询、并发计数、升级检查）验证跨章能力。
- 评分标准帮助你发现薄弱环节，而非追求一次满分。

## 典型场景

1. 用户查询链路：缓存 -> 数据源 -> 回填。
2. 并发任务处理：线程池执行 + 结果统计。
3. 版本升级演练：检查清单 + 灰度策略。

## 常见误区

1. 只做“快乐路径”，不测异常分支。
2. 只看结果，不解释方案边界。
3. 不记录复盘，导致同类问题反复出现。

## Java 示例代码（含注释，可直接运行）

**建议文件名：** `Main.java`  
**运行命令：** `javac Main.java && java Main`

```java
import java.util.LinkedHashMap;
import java.util.Map;

public class Main {
    static class ScoreCard {
        private final Map<String, Integer> scores = new LinkedHashMap<>();

        void add(String item, int score) {
            scores.put(item, score);
        }

        int total() {
            int sum = 0;
            for (int v : scores.values()) {
                sum += v;
            }
            return sum;
        }

        void printDetails() {
            scores.forEach((k, v) -> System.out.println(k + "=" + v));
        }
    }

    public static void main(String[] args) {
        // 模拟综合练习自评分
        ScoreCard card = new ScoreCard();
        card.add("概念准确", 22);
        card.add("原理完整", 20);
        card.add("代码质量", 21);
        card.add("风险与验证", 18);

        card.printDetails();
        int total = card.total();

        System.out.println("total=" + total);
        System.out.println(total >= 80 ? "result=PASS" : "result=REVIEW");
    }
}
```

**预期输出示例：**

```text
概念准确=22
原理完整=20
代码质量=21
风险与验证=18
total=81
result=PASS
```

## 练习与自测

1. 完成 3 个综合题并写出方案说明。
2. 每题补 1 个失败分支处理方案。
3. 自测：是否能在 5 分钟内口述“基础层完整技术主线”。

## 本章小结

- 基础层的完成标准是“能串起来、能跑起来、能讲出来”。
- 通过综合训练，进入进阶阶段会更稳。
