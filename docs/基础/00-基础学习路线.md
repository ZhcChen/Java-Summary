# 基础学习路线

## 学习目标

完成基础层后，你应具备以下能力：

- 能写出结构清晰、边界明确的 Java 基础代码。
- 能解释常见基础问题背后的原理，而不是只记结论。
- 能阅读并理解以 Java 8 为主的业务代码，同时知道 9~17 的关键增量。

## 三阶段路线

### 阶段 A：语言与基础库

- `01` 语法与面向对象
- `02` 集合与泛型
- `03` 字符串、数值与常用基础库

### 阶段 B：稳定性基础

- `04` 异常机制与调试思维
- `05` IO/NIO 与文件处理
- `06` 并发基础与线程池
- `07` JVM 基础与内存模型

### 阶段 C：版本演进与综合应用

- `08` Java 8 核心特性
- `09` Java 9~17 关键特性
- `10` 版本升级与兼容实践
- `11` 基础综合练习与自测

## 学习建议

- 每章按“概念 -> 原理 -> 场景 -> 代码 -> 练习”顺序阅读。
- 每章示例代码必须自己跑一遍，再改 1~2 处观察结果。
- 不追求一次记住全部，先建立正确心智模型。
