# Java 版本新特性索引（纯导航）

> 基线：Java 8。后续版本按“新增特性 -> 适用场景”逐步补充。

## Java 8（基线）

- Lambda、Stream、Optional、时间库（`java.time`）

## Java 9 ~ 10

- 模块化（JPMS）
- 局部变量类型推断（`var`）

## Java 11（LTS）

- HttpClient（标准库）
- 字符串与文件 API 增强

## Java 17（LTS）

- Sealed Class
- Pattern Matching（分阶段演进）
- Records（正式可用）

## Java 21（LTS）

- Virtual Threads
- Structured Concurrency（演进特性）
