## 前置知识

- 了解方法调用与执行顺序。
- 会写匿名函数或 lambda。
- 知道程序可以并行执行多个任务。

## 术语解释（零基础友好）

- **线程状态**：线程在不同阶段的运行状态，如就绪、运行、阻塞、终止。
- **join**：让当前线程等待目标线程执行完成。
- **并发**：同一时间段处理多个任务。

## 详细学习步骤（从不会到会）

1. 先创建一个最小线程并输出日志，观察与主线程的执行顺序。
2. 通过 `join` 控制线程协作，理解等待语义。
3. 模拟阻塞与结束状态，观察线程生命周期变化。
4. 总结不同创建方式（Thread、Runnable、线程池）适用场景。

## 常见错误与纠偏

- 创建线程后忘记 `start`，任务实际上未并发执行。
- 滥用手动创建线程，导致线程数量失控。

## Java 示例代码（含注释，可直接运行）


**建议文件名：** `Main.java`  
**运行命令：** `javac Main.java && java Main`

**预期输出（示例）：**
```text
worker running
main done
```

```java
public class Main {
    public static void main(String[] args) throws InterruptedException {
        Thread worker = new Thread(() -> {
            // 子线程执行任务
            System.out.println("worker running");
        });

        worker.start();
        // join 确保主线程等待子线程结束
        worker.join();
        System.out.println("main done");
    }
}
```
