# 按学习顺序索引

> 目标：从“能写业务”到“能讲原理”再到“能做架构权衡”。

## 推荐顺序

```mermaid
flowchart LR
    A["阶段 1: L1 基础夯实"] --> B["阶段 2: L2 工程与性能"]
    B --> C["阶段 3: L3 架构与治理"]
    C --> D["阶段 4: 面试冲刺"]
```
