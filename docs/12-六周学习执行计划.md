# 六周学习执行计划（可直接执行）

> 目标：6 周内完成从 L1 到 L3 的一轮系统复习，并形成可口述面试输出。

## 总体节奏

```mermaid
flowchart LR
    A[第1-2周 L1基础] --> B[第3-4周 L2进阶]
    B --> C[第5周 L3架构]
    C --> D[第6周 综合实战]
```

## 每周固定动作

- 学习：5 天（每天 1.5~2 小时）
- 复盘：1 天（口述 + 错题）
- 模拟：1 天（限时问答 + 系统设计）

- P0 题可稳定口述 80% 以上。
- 每个层级至少 1 套完整答题模板。
- 能在 20 分钟内完成一题系统设计阐述。
